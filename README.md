# pili-short-video-kit

## 1. 概述

`pili-short-video-kit` 是七牛开发的一款集音视频采集、处理、压缩、上传、播放于一体的 SDK，接口简单易用，支持添加滤镜、字幕、MV、背景音乐、动图等特效处理，以及视频编辑等功能。

## 2. 目录结构

### 2.1 Android

该目录下为 Android SDK 和 demo 工程代码。

### 2.2 iOS

该目录下为 iOS SDK 和 demo 工程代码。

### 2.3 Common

该目录下为一些公共的代码和脚本。
